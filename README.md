# ⚡ DeaSint // OSINT TOOL

`DeaSint` — это легковесный автономный инструмент на базе Python и графической библиотеки Tkinter. Он разработан для удобного структурирования, локального хранения, компиляции и экспорта собранных OSINT-данных (досье) по трем основным категориям: физические лица, организации и транспортные средства.

> **ВАЖНОЕ ПРИМЕЧАНИЕ:** Этот инструмент является полностью локальным. Он не содержит встроенных сетевых парсеров, не выполняет автоматический поиск по интернету и не отправляет вводимые вами данные на внешние сервера или сторонние ресурсы. Вся работа происходит исключительно на вашем компьютере.


## 🎯 Основной функционал

* **Target Individual Specification:** Модуль для систематизации данных о физических лицах (ФИО, никнеймы, контакты, адреса, цифровой след).
* **Corporate Entity Specification:** Форма для анализа юридических лиц, их доменов, поддоменов и MX-инфраструктуры.
* **Vehicle Asset Specification:** Удобное структурирование данных по транспортным средствам (марка, госзнаки, VIN-коды, СТС/ПТС).
* **Doxbin-Style Compiler:** Автоматическая сборка всех заполненных полей в один кастомизированный текстовый файл `.txt` с красивым ASCII-артом и разделением блоков.
* **Динамические стили:** Поддержка визуальных тем оформления интерфейса (Dracula, Cyberpunk, Matrix Green, Classic Dark).
* **Anti-Boredom Protocol:** Встроенная прямо в боковую панель мини-игра "Chrome Dino", позволяющая переключиться и отдохнуть во время рутинной работы с информацией.


## 💻 Инструкция по установке и запуску (Instruction)

### Вариант 1. Запуск скомпилированной версии (.EXE)
1. Перейдите в раздел **Releases** справа на главной странице этого репозитория.
2. Скачайте архив `DeaSing - Tool.zip` из последнего доступного релиза.
3. Распакуйте архив в любую удобную папку на вашем компьютере.
4. Запустите файл `DeaSint.exe`. Установка Python или дополнительных библиотек не требуетсяю
